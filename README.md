### Jekyll instructions

Installing Jekyll can be tricky as it depends on ruby which is older on mac. 

* Install asdf using [instructions](https://asdf-vm.com/guide/getting-started.html)

* Use it to install a ruby version
```
asdf install
```

* Restart the shell

* Run jekyll
```
$ jekyll serve
```
